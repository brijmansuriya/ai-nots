# Extension Icons

Place your extension icons in this directory:

- `icon16.png` - 16x16 pixels
- `icon32.png` - 32x32 pixels  
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

You can generate these icons from a single image using tools like:
- [Favicon Generator](https://favicon.io/)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- [Icon Generator](https://www.favicon-generator.org/)

For now, you can use placeholder icons or create simple colored squares with the text "AI" or your logo.
